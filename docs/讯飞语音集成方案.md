# 科大讯飞语音集成方案 — 爸妈宝 (voice-sdk-skill)

> 版本: v1.0 | 日期: 2026-06-29 | 状态: 方案定稿

---

## 一、项目背景

爸妈宝（老人用药提醒 APP）两大核心交互均依赖语音能力：

1. **口服药记录** — 老人口述 "今天早上吃了阿莫西林一粒"，ASR 转文字录入
2. **语音提醒播报** — "该吃降压药了" 定时 TTS 播报

## 二、语音场景与讯飞产品映射

| 场景 | 输入/输出 | 讯飞能力 | SDK类型 | 网络要求 |
|------|-----------|---------|---------|---------|
| 口述用药记录 | 语音→文字 | 语音听写(ASR) | 听写SDK | 在线/离线均可 |
| 服用确认 | 语音→文字 | 命令词识别 | 离线命令词SDK | 离线可用 |
| 服药定时播报 | 文字→语音 | 在线语音合成(TTS) | 语音合成SDK | 在线 |
| 语音问答"我今天吃了什么" | 语音→文字→TTS | ASR+NLU+TTS组合 | 语音理解SDK | 在线 |
| 页面文字朗读 | 文字→语音 | 在线语音合成(TTS) | 语音合成SDK | 在线 |
| 方言口述(湖南话) | 语音→文字 | 方言识别(湘语) | 听写SDK | 在线 |

## 五、MethodChannel 接口定义

### 5.1 Dart → Native (调用)

```dart
// VoiceService MethodChannel 'com.bamaobao/voice'

// 初始化
_invoke('init') → bool

// 设置参数
_invoke('setParams', {
  'vad_timeout': 2000,     // 静音检测超时ms
  'language': 'zh_cn',
  'accent': 'mandarin',     // 或 'cantonese' / 'sichuan'
})

// 开始在线语音识别
_invoke('startListening') → String  // 识别结果文本

// 开始离线命令词识别
_invoke('startOfflineListening') → String  // 匹配的命令词

// TTS 播报（回调通知播放完成）
_invoke('speak', {'text': '该吃降压药了'}) → void

// 停止
_invoke('stopListening') → void
_invoke('stopSpeaking') → void

// 销毁
_invoke('destroy') → void
```

### 5.2 Native → Dart (事件回调)

```dart
// 识别结果
_channel.invokeMethod('onResult', {'text': '我吃了阿莫西林'});

// 错误
_channel.invokeMethod('onError', {'code': 10118, 'message': '网络不可用'});

// 状态变化
_channel.invokeMethod('onStatus', {'status': 'listening'});

// 音量回调
_channel.invokeMethod('onVolume', {'volume': 0.8});

// TTS 播放完成
_channel.invokeMethod('onSpeakCompleted', {});
```

在 `AndroidManifest.xml` 添加：

```xml
<meta-data
    android:name="com.iflytek.appid"
    android:value="YOUR_APPID" />
```

线下编译需要 keystore，用于讯飞 SDK 鉴权。  
**去华为注册开发者账号时会同时生成签名指纹，届时一并提交讯飞控制台。**

## 十、Flutter 端 Mock 方案（先出Demo）

在讯飞 APPID 未注册前，用本地 Mock 替代：

```dart
class MockVoiceService {
  Future<String> startListening() async {
    await Future.delayed(const Duration(seconds: 2));
    return '帮我看看今天的药';  // 模拟识别
  }
  
  Future<void> speak(String text) async {
    print('[Mock TTS] $text');
  }
}
```

启动阶段 VoiceService 通过 `kReleaseMode` 判断：
- Debug/Mock: 用 MockVoiceService
- Release: 用真实 VoiceService
