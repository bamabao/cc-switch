# 药品推荐逻辑校验规则文档

> 交付物 2/4 · 小电 ⚡ · 2026.6.29
> 适用：爸妈宝 MVP v1 药品管理模块

---

## 一、文档目的

本文档定义药品推荐和替换逻辑的校验规则，供后端（刘工）实现时参考，同时作为业务侧验收标准。

---

## 二、核心概念

### 2.1 药品四要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 通用名 | 药品有效成分的法定名称 | 氨氯地平 |
| 商品名 | 药企注册的商标名 | 络活喜、安内真 |
| 剂型 | 药品形态 | 片剂、胶囊、缓释片 |
| 规格 | 每单位含量 | 5mg/片、10mg/片 |

### 2.2 替换药 vs 等效药

- **等效药：** 通用名相同、规格相同、剂型相同 → 可直接替换
- **同类替换药：** 通用名不同但药理作用相似 → 需医生确认
- **不可替换：** 成分、作用机制完全不同 → 禁止替换

---

## 三、替换推荐规则

### 3.1 规则一：等效药可直接替换

```
条件：通用名相同 AND 规格相同 AND 剂型一致（片剂=片剂，胶囊≠片剂）
结果：可替换 ✅，推荐所有符合条件药品，按价格排序
```

**示例：**
- 络活喜（氨氯地平 5mg 片剂）→ 安内真（氨氯地平 5mg 片剂）✅ 可替换
- 络活喜（氨氯地平 5mg 片剂）→ 压氏达（氨氯地平 5mg 片剂）✅ 可替换

### 3.2 规则二：规格不同提示医生确认

```
条件：通用名相同 AND 规格不同 AND 剂型相同
结果：可替换但需调整用量 ⚠️，需标明换算关系
```

**示例：**
- 氨氯地平 5mg/片 → 氨氯地平 2.5mg/片 ⚠️（每次需吃2片）

### 3.3 规则三：剂型不同需谨慎

```
条件：通用名相同 AND 剂型不同（如片剂 vs 缓释片）
结果：警告 🔴 不可自行替换，需遵医嘱
```

**示例：**
- 硝苯地平片 → 硝苯地平缓释片 🔴（释放速度不同，严禁自行替换）

### 3.4 规则四：同类药提示医生评估

```
条件：通用名不同 AND 属于同一药理分类
结果：标记"同类药，需医生评估" 🟡
```

**常见同类药组：**

| 类别 | 通用名列表 |
|------|-----------|
| CCB类降压药 | 氨氯地平、硝苯地平、非洛地平、左氨氯地平 |
| ACEI类降压药 | 卡托普利、依那普利、贝那普利、培哚普利 |
| ARB类降压药 | 缬沙坦、厄贝沙坦、替米沙坦、氯沙坦 |
| 他汀类降脂药 | 阿托伐他汀、瑞舒伐他汀、辛伐他汀 |
| 双胍类降糖药 | 二甲双胍（普通片/缓释片/肠溶片）|
| 磺脲类降糖药 | 格列美脲、格列齐特、格列吡嗪 |

### 3.5 规则五：不同类药物禁止替换

```
条件：通用名不同 AND 药理分类不同
结果：🚫 不可替换
```

**示例：**
- 氨氯地平（降压药）→ 二甲双胍（降糖药）🚫
- 阿莫西林（抗生素）→ 布洛芬（止痛药）🚫

## 四、禁忌搭配规则

### 4.1 禁止搭配（同时服用有严重风险）

| 药品A | 药品B | 风险说明 |
|-------|-------|---------|
| 华法林 | 阿司匹林 | 出血风险显著增加 🩸 |
| 华法林 | 布洛芬 | 同上，NSAIDs增加抗凝效果 |
| 氯吡格雷 | 奥美拉唑 | 降低氯吡格雷药效 |
| ACEI类 | 螺内酯 | 高钾血症风险 ⚡ |
| 二甲双胍 | 碘造影剂 | 需停用48小时，预防乳酸酸中毒 |
| 他汀类 | 克拉霉素 | 增加横纹肌溶解风险 |
| 硝苯地平 | 葡萄柚汁 | 大幅增加血药浓度 |

### 4.2 谨慎搭配（需监测、调整剂量）

| 药品A | 药品B | 注意事项 |
|-------|-------|---------|
| 降压药 × 2种以上 | — | 血压过低风险，建议监测血压 |
| 降糖药 + 降压药 | — | 部分降压药可能影响血糖 |
| NSAIDs + 降压药 | — | 降低降压药效果 |
| 利尿剂 + NSAIDs | — | 降低利尿效果，增加肾毒性 |

### 4.3 互动校验逻辑

```
输入：用户当前所有活跃药品列表
输出：逐对检查 → 匹配禁止搭配表 → 生成警告列表

优先级：
  🔴 禁止搭配 → 阻止添加，提示就医
  🟡 谨慎搭配 → 允许添加但展示警告
  🟢 安全组合 → 无提示
```

## 五、用药方案合理性校验

### 5.1 剂量上限校验

| 药品 | 单次上限 | 每日上限 | 来源 |
|------|---------|---------|------|
| 对乙酰氨基酚 | 500mg | 2000mg | 中国药典 |
| 布洛芬 | 400mg | 1200mg | 说明书 |
| 阿莫西林 | 1000mg | 4000mg | 说明书 |
| 阿司匹林（抗血小板）| 100mg | 100mg | 临床指南 |
| 氨氯地平 | 10mg | 10mg | 说明书 |

### 5.2 服用时间校验

```
规则：
  早餐后时段（07:00-09:00）：适合需与食物同服的药
  睡前时段（21:00-23:00）：适合他汀类（晚间合成胆固醇）
  空腹时段：部分降糖药、甲状腺素

校验逻辑：
  如用户将"阿托伐他汀"设为早餐后 → 提示：他汀类建议睡前服用效果更佳
  如用户将"左甲状腺素"设晚餐后 → 提示：建议早起空腹服用，避免食物影响吸收
```

### 5.3 漏服处理规则

| 药品类型 | 漏服处理方式 |
|---------|-------------|
| 每日1次 | 当天想起立即补服；次日想起跳过，正常服用 |
| 每日2-3次 | 想起时间距下次服药 ≥4小时 → 补服；否则跳过 |
| 降压药 | 当日可补，但勿双倍剂量 |
| 降糖药 | 未进食不能补服降糖药 |
| 抗生素 | 想起立即补服，下次服药时间顺延 |
| 激素类 | 当日立即补服，勿跳过 |

**系统提示文案：**

| 场景 | 文案 |
|------|------|
| 当日漏服可补 | 「%s」今天还没吃，现在补上，下次按时吃就好 |
| 当日漏服跳过 | 「%s」今天漏了，不用补了，明天按时吃。**切勿加倍剂量！** |
| 抗生素漏服 | 「%s」漏服了，立即补上，下次服药时间顺延 |

## 六、数据字典字段

### 药品表（medicine）

```json
{
  "genericName": "氨氯地平",       // 通用名（核心匹配字段）
  "brandName": "络活喜",           // 商品名
  "category": "CCB类降压药",       // 药理分类
  "dosageForm": "片剂",            // 剂型
  "specification": "5mg",          // 规格
  "manufacturer": "辉瑞",          // 厂商
  "atcCode": "C08CA01"            // ATC分类编码（可选，用于精确匹配）
}
```
