# 健康日报内容模板

> 交付物 3/4 · 小电 ⚡ · 2026.6.29
> 适用：爸妈宝 APP 健康日报模块

---

## 一、模块定位

> 「健康日报」是让老人每天打开 APP 的**核心理由**。纯用药提醒太冷，加入天气、养生内容后变成老人愿意主动看的产品。

**日报三大板块：**
1. 📋 今日用药小结
2. 🌤️ 天气提醒
3. 💡 养生小贴士

---

## 二、今日用药小结

### 模板逻辑

```
输入：今日药品总数 N、已服用数 M
输出：根据完成率选择对应文案
```

### 模板内容

**完成率 100% (M = N)**
> 🎉 今天 %d 种药都按时吃了，表现很棒！奖励自己一杯温水 🥛

**完成率 ≥ 50% (M ≥ N/2)**
> 👍 今天吃了 %d 种，还有 %d 种别忘了哦，加油！

**完成率 > 0% (M < N/2)**
> ⏰ 今天才吃了 %d 种，还有 %d 种该吃了

**完成率 0% (M = 0)**
> 💊 今天还没开始吃呢，现在开始也不晚！

### 补药提醒尾部（漏服时追加）

> 💡 漏服补吃指南：%s 类药物漏了可在发现时立即补服，但切勿一次吃双倍剂量！

根据主要药品类型动态填充类名。

## 三、天气提醒

### 数据来源

调用天气 API，获取：天气类型、温度范围、空气质量、特殊预警

### 模板内容

**晴天（最高温 < 30°C）**
> 🌤️ 今天多云转晴，%s~%s°C，不冷不热刚刚好。午后散步半小时，晒晒太阳补补钙。

**晴天（最高温 ≥ 30°C）**
> ☀️ 今天%s~%s°C，气温偏高。上午10点前出门活动，午后减少外出，多喝水防中暑。

**阴天**
> ☁️ 今天%s~%s°C，阴天。光线偏暗，出门走路多看脚下，带件薄外套。

**小雨/中雨**
> 🌧️ 今天有雨，%s~%s°C。地面湿滑尽量少出门，必须出门的话穿防滑鞋，带把伞。

**大雨/暴雨**
> ⛈️ 今天大雨，%s~%s°C。尽量不要出门！必须外出的话避开低洼路段，远离电线杆。

**雪天**
> ❄️ 今天下雪，%s~%s°C。路面结冰千万小心，不出门最安全。记得穿暖和，帽子围巾都带上。

**极端高温（≥ 35°C）**
> 🔥 高温警报！今天 %s°C，老人外出极易中暑。尽量待在有空调的房间，每半小时喝一小杯水。

**极端低温（≤ 0°C）**
> 🥶 寒潮来袭！今天 %s°C，心脑血管风险增加。注意头、颈、脚三个部位的保暖，尽量不出门。

**空气质量差（AQI > 150）**
> 😷 今天空气不太好，污染指数偏高。有心肺疾病的老人尽量不外出，出门务必戴口罩。

**大风天（风力 ≥ 6级）**
> 💨 今天风大，降温明显。出门注意防风，帽子戴紧，远离广告牌和临时搭建物。

### 换季提示（根据月份动态触发）

**3月-4月**
> 🌸 春天早晚温差大，别急着脱厚衣服。"春捂秋冻"，护好脖子和后背。

**6月-8月**
> ☀️ 夏天暑气重，多喝绿豆汤、菊花茶解暑。空调别开太低，26°C 最合适。

**9月-11月**
> 🍂 秋天干燥，多吃梨、银耳、百合润肺。早晚凉了，出门记得添件外套。

**12月-2月**
> 🧣 冬天血管收缩，血压容易波动。按时吃药最关键，出门一定戴帽子围巾！

## 四、养生小贴士

### 模板设计原则

- 每条不超过 50 字，字体放大适配老年人阅读
- 基于用户慢性病标签做个性化推荐
- 每日轮换，避免重复

### 通用养生（无慢性病标签的用户）

| 编号 | 贴士内容 |
|------|---------|
| G-01 | 🚶 每天散步 30 分钟，比吃补药管用。走不动就慢走，动就比不动强。 |
| G-02 | 💧 每天喝够 6-8 杯水。不要等渴了再喝，那时候身体已经缺水了。 |
| G-03 | 😴 午睡 20-30 分钟最好。别睡超 1 小时，反而更累还影响晚上睡眠。 |
| G-04 | 🥗 每顿饭七分饱，细嚼慢咽，胃肠没负担。 |
| G-05 | 🥛 早晚一杯温牛奶，补钙又助眠。 |
| G-06 | 🧠 多跟人聊天、打打牌、看看书，脑子越用越灵光。 |
| G-07 | 🦷 饭后漱口、早晚刷牙。牙好，胃口才好，身体才好。 |
| G-08 | 🛁 泡脚水温 40°C 左右、20 分钟以内，微微出汗正好。糖尿病人要特别当心水温。 |
| G-09 | 🌞 每天晒 15-20 分钟太阳（避开正午），帮助身体合成维生素D。 |
| G-10 | 😊 心态要松。少生气、少操心，儿孙自有儿孙福。 |

### 高血压专属

| 编号 | 贴士内容 |
|------|---------|
| H-01 | 🧂 每天盐不超过一小勺（5克）。酱油、咸菜、腊肉都是"隐形盐"。 |
| H-02 | 📏 每天早晚测血压，同一时间、同一手臂、同一姿势，才准。 |
| H-03 | 🏃 快走、太极拳、慢骑自行车都行，每周至少 5 次、每次 30 分钟。 |
| H-04 | 🚫 降压药不能自己停！血压正常了≠病好了，擅自停药很危险。 |
| H-05 | 🍌 多吃香蕉、土豆、菠菜等含钾食物，有助降血压。 |

### 糖尿病专属

| 编号 | 贴士内容 |
|------|---------|
| D-01 | 🍚 主食粗细搭配：白米饭里加点糙米、燕麦、杂豆。 |
| D-02 | 🍬 随身带几颗糖或一小包饼干，感觉心慌、出冷汗马上吃。 |
| D-03 | 🦶 每天洗脚后仔细检查脚底，有破皮、水泡及时处理，别拖。 |
| D-04 | 🍉 水果可以吃，但要挑时候——两餐之间吃，一次别超一个拳头大。 |
| D-05 | 📊 每周至少测 1-2 次空腹和餐后血糖，记下来给医生看。 |

### 冠心病专属

| 编号 | 贴士内容 |
|------|---------|
| C-01 | 🚶 运动要匀速，别突然剧烈。走路累了就停下休息，别硬撑。 |
| C-02 | 😰 胸闷、胸痛超过 5 分钟不缓解，立刻打 120，别自己开车。 |
| C-03 | 💊 硝酸甘油要随身带，感觉不舒服舌下含一片，5分钟不缓解含第二片。 |
| C-04 | 🍖 少吃肥肉、动物内脏、油炸食品，血脂控制好，血管才通畅。 |
| C-05 | 🥶 冬天早晨是心脏病高发时段，锻炼等太阳出来再出门。 |

### 高血脂专属

| 编号 | 贴士内容 |
|------|---------|
| L-01 | 🥜 每天一小把坚果（核桃、杏仁），不饱和脂肪酸对血管好。 |

| L-03 | 🍳 炒菜少放油，蒸、煮、凉拌更健康。 |

| 编号 | 贴士内容 |

| O-02 | 🏠 家里地毯防滑、浴室加扶手、夜灯常亮——防摔就是防骨折。 |
| O-03 | 🏋️ 力所能及做点负重运动（提轻物、慢走），刺激骨骼生长。 |
