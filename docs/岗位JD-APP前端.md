# 🧓 岗位 JD：APP 前端开发工程师（Flutter 方向）

## 项目简介

**爸妈宝** — 专注老人用药管理的智慧化平台。本岗位负责**老人端 APP**（Flutter）的前端开发，服务于中老年用户群体，需要深度的**适老化设计**经验。

---

## 一、岗位职责

1. **APP 主体架构与功能开发**
   - 基于 Flutter 框架搭建老人端 APP 整体架构
   - 实现药品录入、查看、搜索等核心功能模块
   - 开发用药提醒系统：**超大弹窗 + 语音播报**，确保老人能及时看到 / 听到服药通知
   - 开发积分商城模块（浏览、积分兑换、订单记录）

2. **适老化 UI/UX 落地**
   - 落地**超大字体**（最低 18sp 起，主信息 24sp+）、**高对比度色彩**（WCAG AA 级）
   - 简化操作路径，单页面核心操作不超过 3 个按钮
   - 支持**语音输入**替代键盘文字录入（药品名称、剂量等）

3. **后端联调与集成**
   - 对接已上线的 Python FastAPI 后端（20+ API 端点），处理认证、数据上报、推送配置
   - 集成语音 SDK（如百度语音 / 讯飞 / 阿里云语音识别）

4. **性能与兼容性**
   - 适配 Android 8.0+ / iOS 13+ 主流机型
   - 针对低端机型的性能优化（启动速度、内存占用、包体积）

## 二、任职要求

### 硬性要求
- 2 年以上 Flutter 开发经验，有**上线交付**的 Flutter APP 项目
- 熟练 Dart 语言，掌握状态管理（Provider / Riverpod / Bloc 至少一种）
- 熟悉 RESTful API 对接及 JSON 数据解析
- 有 Android 或 iOS Native 开发基础（至少熟悉一端打包上架流程）

### 软性要求
- 对**适老化设计**有真实理解和实践（非"把按钮放大"的表面理解）
  - 如：避免手势冲突（防止右滑误触）、关键操作二次确认、字号与行高关系
- 良好沟通能力，能独立与后端工程师协同排查问题
- 有上线 App Store + 各大 Android 应用市场的正式包经验
