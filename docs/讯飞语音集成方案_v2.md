## 二、集成步骤（细化到文件级）

### Step 1: 注册讯飞开发者
1. 打开 https://console.xfyun.cn/
2. 用公司主体注册：岳阳市奥云科技有限公司
3. 创建应用 → 选择"Android"平台
4. 勾选服务：语音听写、语音合成、离线命令词

### Step 2: 获取APPID + APISecret
- 控制台 → 我的应用 → APPID：**2d77802d**（✅ 已获取 2026-06-30）
- 已填入 VoicePlugin.kt 注释中，下载SDK后解注即可

### Step 3: 配置Android签名指纹
```bash
# 用debug keystore生成SHA1
keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
```
- 将SHA1填入讯飞控制台

### Step 4: 下载MSC SDK文件
从讯飞控制台 → SDK下载 → 选择"语音听写+语音合成+离线命令词"

需放入项目的文件清单：

| 文件 | 目标路径 | 说明 |
|------|---------|------|
| `libs/msc.jar` | `android/app/libs/msc.jar` | 讯飞核心库 |
| `libs/armeabi-v7a/libmsc.so` | `android/app/src/main/jniLibs/armeabi-v7a/` | ARM32架构 |
| `libs/arm64-v8a/libmsc.so` | `android/app/src/main/jniLibs/arm64-v8a/` | ARM64架构 |
| `libs/x86_64/libmsc.so` | `android/app/src/main/jniLibs/x86_64/` | 模拟器用 |
| `assets/iflytek/` | `android/app/src/main/assets/iflytek/` | 资源文件 |

### Step 5: 启用真实SDK（解注代码）
在 `VoicePlugin.kt` 中：

```kotlin
// 1) 初始化 — 把Mock注释去掉，启用：
SpeechUtility.createUtility(applicationContext, SpeechConstant.APPID + "=$appId")
speechRecognizer = SpeechRecognizer.createRecognizer(applicationContext)
synthesizer = SpeechSynthesizer.createSynthesizer(applicationContext, null)

// 2) handleStartListening → 用真实 listener 替换 Mock
speechRecognizer?.startListening(recognizerListener)

// 3) handleSpeak → 用真实 synthesizer 替换 Mock
synthesizer?.startSpeaking(text, synthesizerListener)
```

### Step 6: build.gradle.kts 配置
在 `android/app/build.gradle.kts` 添加：
```kotlin
android {
    sourceSets {
        getByName("main") {
            jniLibs.srcDirs("src/main/jniLibs")
        }
    }
}
```

### Step 7: proguard 配置
```pro
# proguard-rules.pro
-keep class com.iflytek.** {*;}
-keepattributes Signature
```
