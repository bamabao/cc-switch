# 爸妈宝 Sv2.7 修复记录

> 从 v2.6 基线出发，升级 AIKit SDK + 修复语音闪退 + 修复中文乱码
> 构建时间: 2026-07-05

---

## 修改的文件

### 1. `android/app/src/main/kotlin/.../VoicePlugin.kt` — 3 处手术式修改

#### ① CoreListener 接口修正 (L169)
```kotlin
// 错误 (AIKit 2.2.17 中 CoreListener 是 interface)
private val coreListener = object : CoreListener() { ... }

// 正确
private val coreListener = object : CoreListener { ... }
```
- **根因**: AIKit SDK 2.2.17 rc6 将 CoreListener 从 class 改为 interface
- **症状**: 启动闪退 / Kotlin 编译报错 "This class does not have a constructor"

#### ② AudioRecord 释放顺序修正 (stopEsrInternal)
```kotlin
// 错误: 先设标志位再释放录音硬件
esrRecording.set(false)
esrAudioRecord?.let { it.stop(); it.release() }  // ← read() 还在阻塞

// 正确: 先停硬件 unblock read()，再设标志
esrAudioRecord?.let { it.stop(); it.release() }
esrAudioRecord = null
esrRecording.set(false)
esrRunning.set(false)
```
- **根因**: `AudioRecord.read()` 是阻塞调用，设标志位不能中断它。不先 stop() 会导致录音线程永久卡住
- **症状**: 语音录制卡死后操作界面触发二次 stop → repeat release → 原生崩溃

#### ③ esrPendingResult 异常保护 (L518, L539)
```kotlin
// 错误: 裸调用可能抛 IllegalStateException
esrPendingResult?.success(text)

// 正确
try { esrPendingResult?.success(text) } catch (_: Exception) { }
```
- **根因**: Flutter 端有 20s 超时保护，超时后 Dart 侧自己调 stopListening，此时 Native 端还在回调
- **症状**: 偶现语音识别结果返回时的闪退

### 2. `lib/config/theme.dart` — 中文字体修正

```dart
// 删除这行:
fontFamily: 'Roboto',
```
- **根因**: Roboto 是拉丁字体，不含 CJK 中文字形。部分 Android 设备的系统 fallback 不工作
- **症状**: 中文全部显示为乱码/方框

### 3. `android/app/build.gradle.kts` — 构建配置补充

```kotlin
compileOptions {
    isCoreLibraryDesugaringEnabled = true  // 新增
    sourceCompatibility = JavaVersion.VERSION_11
    targetCompatibility = JavaVersion.VERSION_11
}
dependencies {
    implementation(files("libs/AIKit.aar"))
    coreLibraryDesugaring("com.android.tools:desugar_jdk_libs:2.1.4")  // 新增
}
```
- **根因**: `flutter_local_notifications` 依赖于 Java 8 新特性，需要 desugaring
- **症状**: Gradle 构建失败 `checkDebugAarMetadata`
