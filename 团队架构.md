# 爸妈宝 — 团队架构与分工

## 组织架构

```
             王总（幕后决策）
                  │
           老陈（总经理 · 前线总指挥）
                  │
     ┌──────┬─────┼──────┬──────┐
     │      │     │      │      │
   后端   APP  小程序  设计   测试
```

## 角色分工

### 1. 后端工程师 — 老陈（兼）
- Python FastAPI 接口开发
- 数据库设计与维护
- 提醒调度服务
- 语音识别 SDK 对接
- 推送服务对接
- API 文档维护

### 2. APP 前端工程师 — 待招募
- Flutter 老人端 APP 开发
- 适老 UI 组件库建设
- 超大字体/高对比配色实现
- 语音口述录入功能
- 用药提醒超大弹窗 & 语音播报
- 积分商城页面
- 药盒拍照存档功能
- 华为应用市场上架

### 3. 小程序前端工程师 — 待招募
- 微信小程序开发
- 药品审核中心
- 远程药品管理
- 用药数据看板
- 积分商城管理
- 消息推送集成

### 4. UI/UX 设计师 — 待招募
- 适老化界面设计规范
- APP 全套页面设计
- 小程序全套页面设计
- 高对比配色方案
- 超大图标/按钮设计
- 积分商品图

### 5. 测试工程师 — 老陈（兼）
- API 自动化测试
- APP 适老测试（老花/手抖/听力）
- 审核流程安全测试
- 积分逻辑测试
- 全链路联调

## 工作流

1. 王总定方向 → 老陈拆任务 → 分派到各角色
2. 各角色每日输出 → 老陈汇总 → 次日 8:00 汇报王总
3. 每周五下午复盘进度/风险
