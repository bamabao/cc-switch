# 爸妈宝 — 项目开发计划

## 整体排期：4个月（16周）

| 阶段 | 时间 | 交付物 |
|---|---|---|
| **P0 架构 & 数据层** | 第 1-2 周 | 后端骨架、数据库 Schema、API 基础 |
| **P1 药品核心链路** | 第 3-6 周 | 药品录入 → 审核 → 提醒 → 确认 → 记录闭环 |
| **P2 积分商城系统** | 第 7-9 周 | 积分规则引擎、商城、兑换流程 |
| **P3 子女小程序** | 第 10-12 周 | 小程序审核/看板/商城管理/消息推送 |
| **P4 适老界面 & 语音** | 同期 P1-P3 并行 | UI 规范、语音识别兜底、放大界面 |
| **P5 联调测试** | 第 13-14 周 | 全链路联调、适老测试、安全审查 |
| **P6 灰度 & 上线** | 第 15-16 周 | 内测、修 bug、正式发布 |

## 关键里程碑

- **Week 2**: 后端 API 基础可用（用户、药品 CRUD）
- **Week 6**: 药品全链路闭环跑通（录药→审核→提醒→确认→积分）
- **Week 9**: 积分商城可兑换
- **Week 12**: 子女小程序全部功能可测
- **Week 14**: 联调完成，App+小程序+后端三端打通
- **Week 16**: 上线
